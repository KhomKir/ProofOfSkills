Тест кейсы:

1. Проверка доступности функционала создания нового чек-листа.
Шаги:
- На левом сайдбаре нажатиь на кнопку “All courses”
- Перейти на страницу курса RU Scratch - Продвинутый уровень
- В строке "Контрольный урок №1" нажать кнопку "Edit"
Ожидаемый результат:
- появление окна создания нового чек-листа;

2. Проверка возможности добавления нового блока в чек-лист.
Шаги:
- создать новый чек-лист
- нажать на кнопку создания нового блока, отмеченную "+"
Ожидаемый результат:
- появление окна создания нового блока чек-листа;

3. Проверка возможности добавления блока "Тема" в чек-лист.
Шаги:
- создать новый чек-лист
- нажать на кнопку создания нового блока, отмеченную "+"
- нажать кнопку "Тема" в окне создания нового блока
Ожидаемый результат:
- появление нового блока чек-листа;

4. Проверка возможности заполения названия созданного блока "Тема"
Шаги: 
- создать новый блок "Тема" в чек-листе
- внести символы в строку "Добавьте название темы"
Ожидаемый результат:
- введенные символы в строке "Добавьте название темы" сохранились;

5. Проверка возможности добавления блока "Вопрос" в чек-лист.
Шаги:
- создать новый чек-лист
- нажать на кнопку создания нового блока, отмеченную "+"
- нажать кнопку "Вопрос" в окне создания нового блока
Ожидаемый результат:
- появление нового блока чек-листа;

6. Проверка возможности заполнения текстовых полей блока "Вопрос" в чек-лист.
- создать новый чек-лист
- нажать на кнопку создания нового блока, отмеченную "+"
- нажать кнопку "Вопрос" в окне создания нового блока
- внести в текстовые поля "Текст вопроса" и "Подсказка" символы
Ожидаемый результат:
- введенные символы в текстовых полях "Текст вопроса" и "Подсказка" сохранились;

7. Проверка возможности удаления блоков из чек-листа.
- зайти в созданный чек-лист
- нажать на кнопку удаления в виде корзины, возникающую при наведении на блок
Ожидаемый результат:
- блок исчезает из структуры чек-листа

8. Проверка возможности копирования блоков в чек-листе.
- зайти в созданный чек-лист
- нажать на кнопку копирования в виде двух пустых листов, возникающую при наведении на блок
Ожидаемый результат:
- блок копируется и создется второй такой же внутри открытого чек-листа

9. Проверка возможности сохранения созданного чек-листа.
Шаги:
- Создать новый чек-лист
- Добавить блоки в чек-лист и заполнить их текстовые поля
- Сохранить чек-лист, нажав кнопку "Save"
- Перезагрузить страницу курса
- Проверить отображение созданного чек-листа на странице курса
Ожидаемый результат: 
- Сохраненный чек-лист корректно отображается в статусе сохранен после перезагрузки страницы

10. Проверка возможности публикации созданного чек-листа.
Шаги:
- Создать новый чек-лист
- Добавить блоки в чек-лист и заполнить их текстовые поля
- Опубликовать чек-лист, нажав кнопку "Publish"
- Перезагрузить страницу курса
- Проверить отображение созданного чек-листа на странице курса
Ожидаемый результат: 
- Сохраненный чек-лист корректно отображается в статусе опубликован после перезагрузки страницы
